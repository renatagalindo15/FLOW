Configure em 5 min

Siga esses passos para ativar a captação de leads gratuitamente usando Google Forms + Google Drive:

  1. 1 Faça upload do PDF no Google Drive → clique com botão direito → "Compartilhar" → "Qualquer pessoa com o link" → copie o link.
  2. 2 Crie um Google Form com os campos: Nome, Email, Instituição, WhatsApp. Em "Respostas" → ative "Planilha" para coletar tudo no Sheets.
  3. 3 Neste arquivo HTML, substitua https://drive.google.com/file/d/1yhtRWJRBTuNy1LyrWQGIFLnpZHAl-3YS/view?usp=sharing pelo link do Drive, e https://docs.google.com/forms/u/0/d/e/1FAIpQLScqJwkVKT7yFvE51fBjLqGdQP7kgdvbwnCFhRTSqkRH89RZ-g/formResponse pelo link de envio do formulário.
  4. 4 Hospede gratuitamente no GitHub Pages (grátis, sem limite) ou compartilhe diretamente este arquivo HTML via link do Google Drive com permissão pública.
